== Description ==

This plugin allows you link your blog to the Canalplan AC website. You can import routes from the route planner and link your blog posts to the canalplan azetteer.d.

NOTE : You have to have PHP V5 installed for this plugin to work as the code contains several PHP V5 specific features.


== IMPORTANT ==  

This plugin creates 9 tables in your database which occupy about 6MB of space.

The plugin also uses the fopen and file_get_contents calls. Please ensure that these functions are available and that they can access external sites.


== Installation ==

1. [Download] (http://wordpress.org/extend/plugins/canalplan-ac/) the latest version of Wordbooker.
1. Unzip the ZIP file.
1. Upload the `canalplan` directory to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress. 
1. Navigate to `Options` &rarr; `Canalplan AC` for configuration and follow the on-screen prompts.


== Features ==

- Imports data from the Canalplan AC website
- Easy linking to Canalplan AC Gazetteer entries
- Easy inclusion of Googlemaps related to Canalplan AC locations
- Easy inclusion of maps of complete waterways or sections of waterways
- Import Planned routes from Canalplan and create a "Cruising Log" of blog entries for the trip
- Bulk export of links to Canalplan AC Gazetteer entries
- Canalplan AC website automatically links back to relevant blog entries from Gazetteer pages.
